# AI Telegram Bot

Telegram-бот на Python с использованием **aiogram 3+** и **OpenAI API**.  
Бот поддерживает память диалога, переключение ролевых режимов через `prompts.json`, сброс истории и генерацию изображений по команде `/image`.

## Возможности

- диалог с LLM через Telegram
- хранение последних сообщений пользователя и ассистента
- поддержка нескольких режимов работы
- переключение режимов через `/mode`
- очистка памяти через `/reset`
- генерация изображений через `/image`
- конфигурация через `.env`
- простая и расширяемая структура проекта
