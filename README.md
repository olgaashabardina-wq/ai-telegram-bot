# AI Telegram Bot

Telegram-бот на Python с использованием **aiogram 3+** и **OpenAI API**.  
Бот поддерживает память диалога, переключение ролевых режимов через `prompts.json`, сброс истории и генерацию изображений по команде `/image`.

## Возможности

- диалог с LLM через Telegram
- хранение последних сообщений пользователя и ассистента
- поддержка нескольких режимов работы
- переключение режимов через `/mode`
- очистка памяти через `/reset`
- генерация изображений через `/image`
- конфигурация через `.env`
- простая и расширяемая структура проекта

## Запуск, демонстрация ролей и обработка ошибок

1. Укажите необходимые переменные в файле `.env`:
   - `BOT_TOKEN`
   - `OPENAI_API_KEY`
   - `MODEL_NAME`
   - `IMAGE_MODEL`
   - `MAX_HISTORY_MESSAGES`
   - `CONTEXT_MESSAGES_LIMIT`

2. Установите зависимости:

```bash
pip install -r requirements.txt

## Запустите бота
python main.py

## Для переключения режимов используется команда:

`/mode`

Бот показывает список доступных ролей и кнопки для выбора режима.

Примеры доступных ролей:
- Обычный помощник
- Помощник разработчика
- Аналитик
- Учитель
- Редактор текстов

## Обработка ошибок и идеи для тестов

В проекте реализована базовая обработка ошибок при работе с текстовой моделью и генерацией изображений.

В качестве дальнейшего улучшения можно добавить:
- обработку сетевых ошибок;
- обработку таймаутов API;
- обработку недоступности модели;
- более подробное логирование;
- базовые тесты на сценарии сбоев.

### Возможные тестовые сценарии
- запуск без необходимых переменных в `.env`;
- ошибка чтения `prompts.json`;
- проверка корректной работы памяти;
- проверка очистки памяти через `/reset`;
- проверка смены роли через `/mode`;
- проверка ошибки при недоступности image API;
- проверка поведения при сетевом сбое.
